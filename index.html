Customizing Icons

This startpage uses Font Awesome icons. You can customize icons for categories and links in Settings.

Step 1: Find an Icon

  1. Go to fontawesome.com/search
  2. Type a keyword in the search box (e.g., "linux", "music", "cloud")
  3. Click the "Free" filter button on the right side to show only free icons
  4. Click on the icon you want to use

Step 2: Copy the Icon Code

  1. After clicking an icon, a popup will appear
  2. Look for the HTML code that looks like: <i class="fa-brands fa-linux"></i>
  3. Copy only the class value — the part inside the quotes
  4. For example, copy: fa-brands fa-linux

Step 3: Use in Settings

  1. Open Settings and go to the Categories or Links tab
  2. Paste the icon code into the icon field (the first input box)
  3. You'll see a live preview of your icon appear

Examples

  • fa-solid fa-code → (solid style)
  • fa-brands fa-github → (brand logo)
  • fa-solid fa-gamepad →
  • fa-brands fa-spotify →

Tip: Use fa-solid for general icons or fa-brands for company/brand logos.
